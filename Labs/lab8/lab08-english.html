Lab 8: Full Vulnerability Assessment on DVWA

Topic 8 — Secure Development Methodologies II

DVWA + Burp Suite + Nikto + CVSS Scoring

Lab Purpose

This is a capstone assessment lab. Students apply all skills learned in previous labs to conduct a structured vulnerability assessment of DVWA — combining manual testing (Burp Suite) and automated scanning (Nikto/ZAP) — and then score each finding using CVSS. This directly reflects the Vulnerability Management process from Topic 8.

Tools Required

DVWA Burp Suite Nikto (Kali) OWASP ZAP (Kali) CVSS Calculator

Planned Tasks

  • Task 1 — Scoping: Define the assessment scope — which DVWA modules will be tested, what is out of scope, what tools will be used.
  • Task 2 — Testing: Run a combination of manual and automated tests against DVWA. Students apply what they learned in Labs 3–7 (Burp Suite, XSS, SQLi, CSRF, Auth attacks, Nikto, ZAP).
  • Task 3 — CVSS Scoring: For each vulnerability found, calculate a CVSS v3 base score using the online CVSS calculator. Justify the score based on Attack Vector, Complexity, Impact, etc.
  • Task 4 — Report: Write a short vulnerability assessment report with: Executive Summary, Vulnerability Table (name, description, CVSS score, evidence), and Recommendations.

Learning Outcomes

  • Conduct a structured vulnerability assessment combining manual and automated techniques
  • Apply CVSS scoring to real vulnerabilities
  • Practice professional security report writing
  • Understand the full Vulnerability Management lifecycle
Instructor Note: This lab is best used as a graded assignment (part of Assignment #2). Students should produce a PDF report. Allow students to use any tool they have learned so far. CVSS calculator is available at nvd.nist.gov/vuln-metrics/cvss/v3-calculator.
