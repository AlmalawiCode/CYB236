Lab 02: Introduction to Web Application Reconnaissance             

                                         

 In this lab, you will practice the information-gathering phase of web application security using only your browser and its built-in Developer Tools against OWASP Juice Shop, a deliberately vulnerable application running locally via Docker on Kali Linux. You will complete four tasks: identifying the technology stack (web server, framework, and security headers), observing live REST API requests, reading and analyzing JSON responses, and inspecting client-side storage (cookies, localStorage, and sessionStorage). At the end, you will generate a PDF report directly from the lab page by entering your name and student ID and clicking "Generate PDF Report." Submit the PDF report through Blackboard. Estimated time: 45–50 minutes.



جمع المعلومات في أمن تطبيقات الويب
في هذا اللاب، ستتدرب على مرحلة جمع المعلومات (Information Gathering) في مجال أمن تطبيقات الويب، وذلك باستخدام متصفح الويب وأدوات المطور (Developer Tools) المدمجة فيه.

سيتم تنفيذ هذا اللاب عمليًا على تطبيق OWASP Juice Shop، والذي يعمل محليًا باستخدام Docker على نظام Kali Linux.
